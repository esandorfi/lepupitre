# Privacy Policy

Le Pupitre is local-first and offline by default.

Code-signing program privacy statement:
"This program will not transfer any information to other networked systems unless specifically requested by the user or the person installing or operating it."

## Product data handling
- Audio recordings, transcripts, feedback artifacts, and workspace data are stored locally.
- No cloud sync is enabled by default.

## User-initiated network actions
- Release/download access from GitHub.
- Optional model download flows explicitly triggered by the operator/user.

## Contact
For privacy/security concerns, use the process in [SECURITY.md](SECURITY.md).
