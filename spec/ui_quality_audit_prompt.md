# UI Quality Audit Prompt

You will make an audit of quality of the UI code base:
- Is it correctly aligned to domain separation?
- Is SQL correctly separated?
- Are patterns state-of-the-art and pragmatic?
- Are business logic patterns coherent?
- Are security boundaries strict and appropriate?
- Are memory, thread, and async optimizations appropriate for fast, strict, and secure desktop use on macOS, Windows, and Linux?
