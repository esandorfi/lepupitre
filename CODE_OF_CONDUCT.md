# Code of Conduct

## Our standard
We want Le Pupitre to be a respectful, inclusive project.

Expected behavior:
- Be respectful and constructive.
- Focus on technical arguments, not people.
- Accept feedback and discuss disagreements professionally.
- Help newcomers understand context and contribution flow.

Unacceptable behavior:
- Harassment, discrimination, or personal attacks.
- Trolling, intimidation, or abusive language.
- Publishing private information without consent.

## Scope
This applies to repository discussions, issues, pull requests, and project communication channels.

## Enforcement
Maintainers may edit, hide, or close content that violates this code and may restrict participation for repeated or severe violations.

## Reporting
Report conduct issues privately via GitHub maintainers (open a private maintainer contact if available) and include links/screenshots when possible.
